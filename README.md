# ⌨️ Auto Typer

클립보드 텍스트를 자동으로 타이핑해주는 도구입니다.

## ✨ 기능
- 텍스트 붙여넣기 창 + 클립보드 자동 붙여넣기
- 타이핑 속도 조절 슬라이더 (5ms ~ 500ms / 글자)
- 시작 딜레이 설정 (1초 ~ 15초, 원하는 창으로 이동할 시간)
- 일시정지 / 재개 / 중단 버튼
- 시스템 트레이 아이콘 (창 닫아도 백그라운드 실행)
- 타이핑 진행률 표시
- 한글 포함 모든 유니코드 지원

---

## 🚀 실행 방법

### 1. 파이썬으로 직접 실행

```bash
pip install -r requirements.txt
python auto_typer.py
```

### 2. .exe로 패키징 (배포용)

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name "AutoTyper" auto_typer.py
```

- `dist/AutoTyper.exe` 생성됨
- 이 파일 하나만 공유하면 파이썬 없이도 실행 가능!

## 💡 사용 방법

1. 앱 실행 후 텍스트 입력 또는 클립보드 붙여넣기
2. 속도와 딜레이 조절
3. **시작** 버튼 클릭
4. 카운트다운 중 타이핑할 창(카카오톡, 메모장 등)으로 이동
5. 자동으로 타이핑 시작!
